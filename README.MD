# Buzzer System 
Basel Al-Dwairi - 20221502065

 Buzzer system that has an on-off switch with a button, and two pots to control the volume and the frequency

[Hand Sketch][2]

[Wokwi Diagram][3]

___
Fetures:
- Button to toggle the system on and off
  - Using Interrupts
- LED to display the system state
- Buzzer
- Volume Pot: Handles the volume of the buzzer
- Delay/ Frequency Pot: Mimics a distance Sensor
